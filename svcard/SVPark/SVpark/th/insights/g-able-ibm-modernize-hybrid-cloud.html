G-Able จับมือ IBM ร่วมจัดงาน Modernize Infrastructure with Hybrid Cloud for AI

ปัญญาประดิษฐ์ หรือ (AI) ได้กลายมาเป็นหนึ่งในตัวแปรสำคัญที่พลิกโฉมการดำเนินงานขององค์กร ในปัจจุบัน เทคโนโลยีนี้ได้ถูกใช้เป็นเครื่องมือที่องค์กรทั่วโลกตั้งเป้าใช้ในการขับเคลื่อนธุรกิจ และสร้างความได้เปรียบทางการแข่งขัน แต่อย่างไรก็ตาม การนำ AI มาใช้งานอย่างมีประสิทธิภาพนั้น ต้องอาศัย โครงสร้างพื้นฐานทางไอทีที่แข็งแกร่ง ซึ่งไม่ได้หมายถึงเพียงการจัดเก็บข้อมูลหรือการประมวลผลอย่างเดียว แต่ยังรวมถึงความสามารถในการรองรับนวัตกรรม และความซับซ้อนของ AI ได้ในระยะยาว โครงสร้างพื้นฐานนึงที่เป็นหัวใจสำคัญของการเปลี่ยนแปลง นั่นคือ Hybrid Cloud โดย Hybrid Cloud ได้ผสานรวมข้อดีของ Public Cloud และ Private Cloud เข้าไว้ด้วยกัน ยืดหยุ่น รวดเร็วและปลอดภัย ด้วยคุณสมบัติเหล่านี้ Hybrid Cloud จึงกลายเป็นโซลูชันที่ตอบโจทย์องค์กรที่ต้องการ

 

G-Able ในฐานะผู้เชี่ยวชาญด้านดิจิทัลโซลูชันอย่างครบวงจร ได้จับมือกับพันธมิตรระดับโลกอย่าง IBM จัดงานสัมนาภายใต้หัวข้อ “Modernize Infrastructure with Hybrid Cloud for AI” โดยได้เชิญวิทยากรซึ่งเป็นผู้เชี่ยวชาญจากทั้ง 2 บริษัทมาให้ข้อมูลด้านโครงสร้างพื้นฐานทั้ง Server, Storage และ Hybrid Cloud รวมไปถึงการแชร์ Use Case ที่เป็นประโยชน์และใช้ได้จริง ที่จะช่วยสนับสนุนให้องค์กรของลูกค้าก้าวเข้าสู่ AI Driven Organization ได้อย่างมีประสิทธิภาพ

 

โดยภายในงานได้รับความสนใจจากลูกค้าเป็นจำนวนมาก และยังได้รับเกียรติจากคุณวรรณา ศฤงคารบริบูรณ์ Senior Executive Vice President of Go to Market  บริษัท จีเอเบิล จำกัด (มหาชน) กล่าวต้อนรับแขกผู้มีเกียรติ และผู้เข้าร่วมงานอย่างอบอุ่น ณ โรงแรมโซฟิเทล กรุงเทพ สุขุมวิท เมื่อวันที่ 28 พฤศจิกายน 2567 ที่ผ่านมา
